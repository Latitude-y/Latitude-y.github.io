    有三种状态：

    • Changes to be committed:已经从工作区add到暂存区的文件

    • Changes not staged for commit:表示工作区，暂时区都存在的file（文件或文件夹），在工作区进行修改或删除，但是没有add到暂存区

    • Untracked files:表示只在工作区有的file（文件或文件夹），也就是在暂时区没有该file
